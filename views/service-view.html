Service refers to the capabilities, features, solutions, etc. that are provided towards the consumer. We will define the consumers and their consumptions in the next steps.

Give your service provider a name to be able to recognize it. The description is optional but can be useful to provide more context.

A service is what the consumer interacts with. For example: an API endpoint may provide multiple capabilities that are used by the consumers to solve different use cases.
